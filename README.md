# inditex-technical-test

## Environment variables
To config correctly the application you must define next environment variables, you have an example in .env.example file

- APP_PORT (if not defined 8080 is the default value)
- DATABASE_CONNECTION
- DATABASE_USERNAME
- DATABASE_PASSWORD
- DATABASE_DIALECT
- DATABASE_TYPE
